# afay-023.github.io

This folder is organized as a direct publishable root for the `afay-023.github.io` GitHub Pages repository.

## Intended repo layout

The following files are meant to live in the root of the repository:

- `index.html`
- `styles.css`
- `script.js`
- `404.html`
- `.nojekyll`

## What this site does

- Acts as the single portfolio link for resume use
- Introduces Lin Jie with a recruitment-oriented AI product positioning
- Links out to:
  - `https://afay-023.github.io/nightingale-demo/`
  - `https://afay-023.github.io/fresh-site-copilot/`

## Recommended deployment workflow

1. Create or open the GitHub repository named `afay-023.github.io`
2. Copy the files from this folder into the root of that repository
3. Commit and push to the default branch
4. In GitHub Pages settings, serve from the root of the default branch

## Notes

- This is a static site and does not require a build step
- `.nojekyll` is included to avoid Jekyll processing surprises
- `404.html` provides a simple recovery path back to the homepage
